## Website ตู้ถ่ายรูป (Photobooth Web Application) 📸

<span>เป็นเว็บไซต์จำลองระบบตู้ถ่ายรูป (Photobooth) ที่พัฒนาด้วย **Next.js** และ **TypeScript**</span>
<br />
<span>ผู้ใช้งานสามารถใช้งานกล้องเว็บแคมเพื่อถ่ายภาพ เลือกรูปแบบกรอบรูป (Frame Layout) และตกแต่งรูปภาพได้แบบอิสระ</span>
<br />
<span>มีการใช้ *library* **react-dnd** สำหรับสร้างระบบลากวาง (Drag & Drop) เพื่อแปะสติ๊กเกอร์ตกแต่งรูปภาพ</span>
<br />
<span>ใช้ **react-color** ในการสร้างเครื่องมือปรับแต่งสีของกรอบรูป และใส่ฟิลเตอร์ (Filter) ให้กับภาพถ่าย</span>
<br />
<span>ใช้ **html2canvas** สำหรับการรวมองค์ประกอบหน้าจอทั้งหมด (รูปถ่าย, กรอบรูป, สติ๊กเกอร์) แล้วประมวลผลออกมาเป็นไฟล์ภาพชิ้นเดียว</span>
<br />
<span>มีการเชื่อมต่อระบบ Backend และ Storage ด้วย **Supabase** สำหรับจัดการรูปภาพสติ๊กเกอร์ และอัปโหลดภาพที่แต่งเสร็จแล้วเพื่อสร้าง **QR Code** สำหรับดาวน์โหลดลงมือถือแบบเรียลไทม์</span>
<br />
<span>มีการใช้ **Valtio** ที่เป็น *library* ของ **JavaScript/TypeScript** ในการจัดการ *state* ข้ามหน้าจอต่างๆ ตั้งแต่เริ่มถ่ายรูปจนถึงหน้าดาวน์โหลด</span>
<br />
<span>เพิ่มลูกเล่นการเปลี่ยนหน้าจอและการเคลื่อนไหวที่ลื่นไหล (Transitions/Animations) โดยใช้ **Framer Motion** และ **AnimatePresence**</span>
